# fernet-app


Your Secret is Safe Here is a web-based application designed to securely encrypt and decrypt text using the Fernet symmetric encryption algorithm It provides a simple and intuitive interface for generating encryption keys protecting sensitive information and recovering encrypted messages with the correct key

Features

Fernet Key
Enter an existing Base64-encoded Fernet key.
Generate a new secure encryption key with the Generate Key button

Message

Input the plain text you want to encrypt or the encrypted text you want to decrypt

Encrypt

Encrypts the entered message using the provided Fernet key and displays the encrypted output

Decrypt

Decrypts the encrypted message using the same Fernet key and restores the original plaintext

Result

Displays the encrypted or decrypted output

Copy

Copies the result to the clipboard for easy sharing or storage




Key Benefits

Clean and user-friendly interface.
Secure key generation with a single click
Fast encryption and decryption of sensitive messages
One-click copy functionality for convenience.
Uses the Fernet encryption standard, which provides both confidentiality and integrity by ensuring that encrypted data cannot be read or modified without the correct key


This tool is ideal for securely protecting confidential messages before storing or sharing them. Since Fernet uses symmetric encryption the same key must be kept safe and used for both encryption and decryption
